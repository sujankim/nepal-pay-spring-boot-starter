Why NepalPay?

⚡

Fast Integration

Add one dependency, set two config keys, inject the client — done in under 5 minutes.

🔐

Secure by Default

Server-side verification, HMAC signature validation, and callback security are built in.

☕

Java Native

Built specifically for Java 21 and Spring Boot 4. No Node.js bridge. No copy-paste scripts.

🧪

Fully Tested

51 automated tests covering success paths, error paths, signatures, and auto-configuration.

📦

Zero Boilerplate

Spring Boot auto-configuration — no @EnableNepalPay, no @Bean, no config class needed.

🌍

Open Source

MIT licensed. Built for Nepal's developer community. Contributions welcome.

✅ With NepalPay
// 3 lines. Type-safe. Tested. Secure.
var response = khaltiClient.initiatePayment(
    KhaltiInitiateRequest.builder()
        .amount(10000L)
        .purchaseOrderId("ORD-001")
        .purchaseOrderName("Premium Plan")
        .build()
);

// Redirect user to payment page
return response.paymentUrl();
